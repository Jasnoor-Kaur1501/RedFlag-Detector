# RedFlag Detector

Paste a message, a text, a situation, or something they said. Get a full red flag analysis — roasted with precision.

## Live Demo

🔗 [jasnoor-kaur1501.github.io/RedFlagDetector](https://jasnoor-kaur1501.github.io/RedFlagDetector)

## What it does

- Paste any text up to 2000 characters
- Scans for 15 known red flag patterns using regex matching
- Roasts each flag with a specific, honest breakdown
- Rates severity: high / medium / low
- Also detects green flags if they're present
- Gives an overall danger score (0–100) with a verdict
- Copy the full report to share
